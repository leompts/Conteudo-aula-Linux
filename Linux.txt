LINUX: é o kernel ou núcleo do sistema operacional, faz a integração entre hardware e sistema operacional. desenvolvido por diversas pessoas e empresas pelo mundo, existindo varias versões do kernel e vários sistemas operacionais criados baseados no linux.

> multitarefa/multiusuario

distribuição ou "distro": personalização do sistema operacional baseado no Linux. tipos: seg info, servidor, pessoal, forense, etc. Ex: Ubuntu (pessoal), Debian, Suse, Fedeora, Red hat (corporativa), Kali


UBUNTU: sistema operacional código aberto desenvolvido pela Canonical, baseado no Linux, muito aceito pela comunidade.


> TERMINAL, SHELL ou KONSOLE: linha de comando para execução de programas. Uso p/automação de tarefas de processos através de comandos.

Doca: barra de tarfas do Ubuntu Linux




***** COMANDOS:

Ctrl+Alt+T >>abre terminal

pwd >>caminho local (pasta pessoal)

ls >>lista arquivos/pastas dentro do diretorio

ls nome da pasta >>lista arquivos dentro da pasta

ls -l >> listar diretorio/arquivos com detalhes

ls nome_diretorio >>lista o conteudo do diretorio nao corrente

cd nome_pasta >>entra na pasta

mkdir nome_pasta>>cria uma pasta dentro do diretorio
mkdir nome_pasta /caminho do diretorio >>idem anterior

cd .. >>sobe para pasta anterior

cd / >>vai para o diretório raiz

cd ~ >>vai para o diretorio pessoal

man ls >>manual do comando

comando --help >>mostra ajuda sobre o comando

history >>exibe historico de programas e comandos usados no terminal

!! >repete o último comando dado no terminal

Ctrl+L >>limpa a tela do terminal
clear >>tb limpa o terminal

mv pasta novo_nome>>renomeia a pasta

mv pasta ~ >>move a pasta para diretorio pessoal

touch >>criar arquivos vazios

cp nome_arquivo diretorio >>copia arquivo
cp nome_arquivo ~/diretorio/ >>copia arquivo p/pasta no diretorio pessoal

clear >>limpa o terminal

rmdir nome_pasta>>remove um diretorio

rm >>remove arquivos de um diretorio

Ctrl+Z >>para o terminal

Ctrl+d >sai do terminal
exit >tb sai do terminal
